# Alpicia Personal Assistant

> The interface for your AI assistant.

## Overview

This is the official web interface for the Alpicia Personal Assistant. Built with a clean, modern aesthetic.

## Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS + clsx + tailwind-merge
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Language**: TypeScript

## Getting Started

1. **Install Dependencies**

   ```bash
   npm install
   ```

2. **Run Development Server**

   ```bash
   npm run dev
   ```

3. **Build for Production**
   ```bash
   npm run build
   ```

## Features

- Real-time chat interface
- Settings management
- Platform integrations
- Activity monitoring
- Voice support
- Knowledge base
